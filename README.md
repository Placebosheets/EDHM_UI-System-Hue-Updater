## **What is EDHM System Hue Updater?**

EDHM System Hue Updater takes system jump data from Elite Dangerous and uses it to change the hues of a custom theme in EDHM-UI. It informs the player of current star system information with visual colours changing on the HUD as they jump, rather than them having to check the Galaxy Map itself. For example:

*Have UI like orbit lines or supercruise POI’s change to the colour of their economy, allegiance or government as you jump into a new system.*

*Change the hud to follow the colour of the powerplay state you’re in.*

*Make your radar or ship hologram warn you of your current systems security level.*

*Turn night vision into an analysis tool that alerts you to a type of faction state.* 




**Current features include:**



* Galaxy map colour codes inform the colour theme of the environmental HUD in each individual system.
* Select internal ship hub features are also given the same options.
* Different overall themes to choose from - accurate, bright, dark, vaporwave or a variable orange theme (for default ED vibes) that all use the galaxy map as their core data source.
* Keep track of all the features of your current system and their corresponding colour codes in-app as you journey across the bubble.


## **Requirements**

Elite Dangerous: Odyssey and the latest version of EDHM_UI found [here](https://github.com/BlueMystical/EDHM_UI).


## **Installation and Use**



* Apply a custom theme in EDHM_UI (wont work with default theme).
* Open EDHM System Hue Updater and point it to your ED Journal folder (usually C:/Users/user/Saved Games/Frontier Developments/Elite Dangerous) and the correct EDHM-ini folder (usually C:/Users/user/EDHM_UI/ODYSS/EDHM/EDHM-Ini) for the custom theme.
* Assign individual UI elements to a data set in the app, or assign external and ship elements all to one data set (keep them off if you want to retain the original colours from your chosen theme)
* Jump away to the next system or hit ‘Apply Now’ to load immediately. (Application must be running for it to work in game).
